## Download LocalProxy

LocalProxy is currently available for macOS. Download <a target="\_blank" href={`${config.url}/download`}>here</a> to get started.

**Note:** Windows and Linux versions are being developed based on demand. Visit our <a href={`${config.url}/download`}>download page</a> to join the waitlist and be notified when they become available.

## Installation

### macOS

1. Download the LocalProxy `.dmg` file
2. Open the downloaded file
3. Drag LocalProxy to your Applications folder
4. Launch LocalProxy from your Applications

On first launch, macOS may ask you to confirm you want to open the app. Click "Open" to proceed.

## First Run

When you first start LocalProxy:

1. **System Proxy Setup**

   - Click "Enable System Proxy" to route traffic through LocalProxy
   - You may need to enter your system password

2. **Certificate Installation**

   - LocalProxy will prompt you to install its CA certificate
   - This is required for HTTPS inspection
   - Follow the on-screen instructions

3. **Basic Configuration**
   - The proxy starts automatically
   - Default extensions are pre-configured
   - Visit the Settings page to customize
