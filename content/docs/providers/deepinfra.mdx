---

## Prerequisites

- LocalProxy installed and running
- A DeepInfra account (create one at [deepinfra.com](https://deepinfra.com))

## Step 1: Create a DeepInfra API Key

1. **Log in to DeepInfra Dashboard**

   - Go to [deepinfra.com/dash/api_keys](https://deepinfra.com/dash/api_keys)

2. **Create a New API Key**
   - Click **Create API Key** and name it (e.g., “LocalProxy Integration”)

**Important**: Copy the API key immediately after creation.

## Step 2: Configure DeepInfra in LocalProxy

1. **Open LocalProxy Settings**

   - Launch LocalProxy and click **Settings** in the sidebar

2. **Navigate to AI Providers**

   - Select **AI Providers**, then click **Add**

3. **Add DeepInfra Provider**

   - From the provider list, select **DeepInfra** (OpenAI-compatible)

4. **Enter Connection Details**

   - **Base URL**: Leave default unless using a proxy (default `https://api.deepinfra.com/v1/openai`)
   - **API Key**: Paste your DeepInfra API key
   - **Model preferences**: Choose a model (e.g., `meta-llama/Meta-Llama-3-8B-Instruct`, `mistralai/Mixtral-8x7B-Instruct-v0.1`)

5. **Save Configuration**
   - Click **Save** and ensure DeepInfra appears in your AI Providers
